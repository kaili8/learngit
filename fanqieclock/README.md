# 🍅 番茄钟 (Pomodoro Timer)

一个简洁优雅的番茄工作法计时器，纯前端实现，无需任何依赖。

## 功能

- 🍅 **专注计时**：25 分钟专注 + 5 分钟短休息循环
- 🌴 **长休息**：每完成 4 个番茄自动进入 15 分钟长休息
- 🎯 **圆形进度环**：直观展示剩余时间，颜色随模式变化
- 🔊 **音频提醒**：计时结束播放三段式提示音
- 🔔 **桌面通知**：支持浏览器 Notification API
- 💾 **状态持久化**：每日番茄计数自动保存到 localStorage
- ⚙️ **自定义时长**：可调整专注/短休息/长休息时长
- ⌨️ **键盘快捷键**：空格(开始/暂停)、R(重置)、S(跳过)
- 📱 **响应式设计**：适配桌面和移动端

## 使用方式

直接在浏览器中打开 `index.html` 即可使用。

```bash
# 或者用任意静态服务器
python -m http.server 8000
# 然后访问 http://localhost:8000
```

## 快捷键

| 按键 | 功能 |
|------|------|
| 空格 | 开始 / 暂停 |
| R | 重置当前计时 |
| S | 跳过当前阶段 |

## 配色

| 模式 | 颜色 | 含义 |
|------|------|------|
| 🔴 番茄红 `#e74c3c` | 专注 | 工作时段 |
| 🟢 绿色 `#2ecc71` | 短休息 | 5 分钟放松 |
| 🔵 蓝色 `#3498db` | 长休息 | 15 分钟深度休息 |

## 技术栈

- 纯 HTML5 + CSS3 + Vanilla JavaScript
- SVG 圆形进度环
- Web Audio API 提示音
- Notification API 桌面通知
- localStorage 状态持久化
